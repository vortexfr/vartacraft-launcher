    Windows affiche un avertissement de sécurité ?

    C'est normal. Windows SmartScreen bloque parfois les nouvelles applications qui ne disposent pas encore d'une réputation établie sur les serveurs Microsoft. Le launcher Vartacraft est 100% sûr et ne contient aucun virus.

    Si vous voyez "Windows a protégé votre PC", suivez ces étapes :

    • Cliquez sur "Informations complémentaires" (lien en bleu sous le message)
    • Un bouton "Exécuter quand même" apparaît en bas de la fenêtre
    • Cliquez dessus pour lancer l'installation normalement
    Vérifier vous-même que le fichier est sain : Vérifier sur VirusTotal

    VirusTotal est un service de Google qui analyse les fichiers avec plus de 70 antivirus simultanément. Téléchargez le .exe et glissez-le sur le site pour confirmer qu'il est propre.

    Dépannage
    Si le launcher ne fonctionne pas correctement, essayez ces solutions

    Supprimer le dossier de jeu

    Si le jeu ne se lance pas ou plante au démarrage, supprimez le dossier .VartacraftGame. Le launcher le recréera automatiquement au prochain lancement.

    1. Appuyez sur Win + R et tapez %appdata% puis validez
    2. Trouvez le dossier .VartacraftGame et supprimez-le
    3. Relancez le launcher — les fichiers seront retéléchargés automatiquement
    %appdata%\.VartacraftGame

    Réinstaller le launcher

    Si aucune autre solution ne fonctionne, désinstallez complètement le launcher puis réinstallez-le depuis cette page.

    1. Ouvrez les Paramètres Windows → Applications
    2. Recherchez Vartacraft Launcher et désinstallez-le
    3. Supprimez également le dossier .VartacraftGame dans %appdata%
    4. Téléchargez et réinstallez le launcher depuis cette page
